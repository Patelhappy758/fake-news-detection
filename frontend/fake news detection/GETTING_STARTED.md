## 📊 Detection Guide

### Text Analysis
```
1. Click "Text" tab
2. Paste news article (min 50 characters)
3. Watch character counter
4. Click "Analyze Content"
5. View results on right side
```

### Image Analysis
```
1. Click "Image" tab
2. Drag & drop image OR click to browse
3. See preview
4. Click "Analyze Content"
5. View results with confidence score
```

**Supported:** JPG, PNG, GIF (max 5MB)

## 🐛 Troubleshooting

### Common Issues & Fixes

**Modal won't close:**
→ Click "Continue as Guest"

**Can't see chatbot:**
→ Look bottom-right corner for icon

**Dark mode button missing:**
→ Top navigation, right side

**History not saving:**
→ Enable localStorage in browser

**Language not changing:**
→ Refresh page, try again

**Images won't upload:**
→ Check file size (<5MB) and format (JPG/PNG/GIF)

### Still Having Issues?
1. Press F12 (open console)
2. Check for red error messages
3. Clear browser cache
4. Try incognito mode
