## 🎪 Special Effects

### Glassmorphism (Navigation)
- Backdrop blur: 10px
- Background: rgba(255, 255, 255, 0.95)
- Creates frosted glass effect

### Gradient Backgrounds
- Linear gradients (135° angle)
- Multiple color stops
- Used in buttons, badges, icons

### Shadow Depths (5 Levels)
```
sm:   Subtle (cards)
md:   Standard (buttons)
lg:   Elevated (hover states)
xl:   High (modals)
2xl:  Maximum (hero elements)
```

### Smooth Transitions
- Fast: 150ms (small changes)
- Base: 200ms (standard)
- Slow: 300ms (major changes)

✅ **Visual hierarchy** (size, weight, color)

✅ **Motion design** (purposeful animation)
✅ **Feedback** (hover, focus, active states)
✅ **Progressive disclosure** (show relevant content)

## 🎬 User Flow Examples

### Flow 1: Text Analysis
1. Land on Home page
2. Click "Start Detection"
3. See Detect page (Text mode active)
4. Paste news content
5. Watch character counter
6. Click "Analyze Text"
7. See loading overlay
8. View results with confidence score
9. Read detailed explanation

### Flow 2: Ask Chatbot
1. Navigate to Chatbot page
2. See welcome message
3. Type question (e.g., "spot fake news")
4. Press Enter
5. See user message appear
6. Watch bot "think" (500ms delay)
7. See bot response with tips
8. Continue conversation

### Flow 3: Image Upload
1. Go to Detect page
2. Switch to Image mode
3. Drag image onto upload area
4. See visual feedback (border change)
5. Preview image with file info
6. Click "Analyze Image"
7. Loading animation plays
8. Results display with verdict
